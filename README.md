## Getting Started

### Requirements

- Windows 10
- Python 3


### Libraries

- win32-setfiletime
```
pip install -r requirements.txt
```

### Executing the program

1. Run the script to bring up a menu of choices.
```
python3 Baseline-Analyzer.py
```
